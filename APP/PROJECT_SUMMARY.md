### 🚀 **Cách chạy:**

1. **Mở Android Studio**
2. **Open Project** → Chọn thư mục `APP`
3. **Sync Project** (tự động hoặc File > Sync)
4. **Run App** (▶️ button hoặc Shift + F10)

### 📱 **Demo Flow:**
1. **Login Screen** → Nhập username/password bất kỳ
2. **Dashboard** → Xem stats theo role
3. **Vehicle List** → Click "Xem xe"
4. **Search/Filter** → Test các tính năng
5. **Vehicle Details** → Click vào xe để xem chi tiết
